jpashop - JPA 연습장
============

> #### JPA를 공부하며 만든 토이 프로젝트입니다.

### 개발툴과 기술스택 및 주요기술

- #### Back-End
    - Spring boot
    - JPA
    - H2
    - lombok

- #### Front-End
    - thymeleaf

- #### 회원

    - 회원 조회
- #### 상품
    - 상품 등록
    - 상품 수정
    - 상품 조회
- #### 주문
    - 상품 주문
    - 주문 내역 조회
    - 주문 취소
